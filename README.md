## 🌟 GIỚI THIỆU TỔNG QUAN

### 🎯 **Tầm nhìn**
**TeachFlow** được sinh ra từ nhu cầu thực tế của ngành giáo dục Việt Nam: giảm tải công việc hành chính cho giáo viên, số hóa quy trình quản lý điểm số, và tạo cầu nối thông tin giữa giáo viên, học sinh và phụ huynh.

### 💡 **Sứ mệnh**
> "Mang công nghệ đến gần hơn với giáo dục, giúp giáo viên tập trung vào chuyên môn thay vì công việc giấy tờ, đồng thời tạo môi trường minh bạch trong đánh giá học tập."

### 🎓 **Đối tượng người dùng**

| Đối tượng | Mô tả | Nhu cầu chính |
|-----------|-------|---------------|
| 👨‍🏫 **Giáo viên** | Người dùng chính của hệ thống | Quản lý lớp, nhập điểm, theo dõi học sinh, xuất báo cáo |
| 👨‍🎓 **Học sinh** | Người theo dõi kết quả học tập | Xem điểm, xem thông tin cá nhân, theo dõi tiến độ |
| 👪 **Phụ huynh** | Người giám sát | Theo dõi kết quả học tập của con em |
| 👨‍💼 **Ban giám hiệu** | Người quản lý | Xem báo cáo tổng quan, thống kê toàn trường |

### 📦 **Luồng dữ liệu**
User Action → UI → ViewModel → Repository → [Local DB | Firebase]
↓
User Interface ← State Update ← ViewModel ← Response

### 🔥 **Firestore Collections Structure**

#### **users** collection
```json
{
  "userId": "string",
  "email": "string",
  "name": "string",
  "role": "teacher|student|parent",
  "avatar": "string (url)",
  "phone": "string",
  "createdAt": "timestamp",
  "classIds": ["array of class IDs (for students)"]
}
classes collection
json
{
  "classId": "string",
  "className": "string",
  "subject": "string",
  "teacherId": "string",
  "teacherName": "string",
  "academicYear": "string",
  "semester": "1|2",
  "studentCount": "number",
  "createdAt": "timestamp",
  "students": [
    {
      "studentId": "string",
      "studentName": "string",
      "studentCode": "string"
    }
  ]
}
grade_columns collection
json
{
  "columnId": "string",
  "classId": "string",
  "columnName": "string",
  "type": "15min|45min|final|assignment",

🔒 Firebase Security Rules
javascript
// Firestore Rules
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    
    // Users collection
    match /users/{userId} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && request.auth.uid == userId;
    }
    
    // Classes collection
    match /classes/{classId} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && 
        resource.data.teacherId == request.auth.uid;
    }
    
    // Grades collection
    match /grades/{gradeId} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && 
        exists(/databases/$(database)/documents/classes/$(resource.data.classId)) &&
        get(/databases/$(database)/documents/classes/$(resource.data.classId)).data.teacherId == request.auth.uid;
    }
  }
}
